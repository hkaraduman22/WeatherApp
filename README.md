# Mini-Dashboard Projesi
Bu proje, Git ile ekip çalışması, modüler kod geliştirme ve çakışma (merge conflict) yönetimini deneyimlemek amacıyla Alper Uğur'un ödev yönergesine göre hazırlanmıştır.

## Ekip Üyeleri

Öğrenci A (Entegrasyon Lideri): Tarık

Öğrenci B (Modül Geliştirici): Yunus Emre

## Deneyimlerimiz

Öğrenci A (Tarık) - Yeni kodu çekerken hissettikleri: Ekip arkadaşım Yunus Emre'nin çakışmaları çözerek repoya gönderdiği güncel versiyonu yerel ortamıma hatasız bir şekilde çekerken, Git'in modüler projelerdeki senkronizasyon ve entegrasyon gücünü uygulamalı olarak deneyimledim.

Öğrenci B (Yunus Emre) - Çakışmayı çözerken hissettikleri: Karşılaştığım Merge Conflict (Çakışma) durumunda Git'in referans işaretlerini kullanarak sorunu manuel olarak çözümlemek, takım çalışmasında kod ezilmelerinin nasıl güvenle önlendiğini anlamamı ve sürece olan hakimiyetimi artırdı.
